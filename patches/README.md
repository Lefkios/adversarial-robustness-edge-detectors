# Patches — Adversarial Patch Tensors

Patch tensors are 80×80 RGB float32 tensors (~76 KB each).  
Generated by `notebooks/2_attacks/Yolo_Patch_Opt.ipynb` via EOT optimisation (§4.3, §5.8).  
Selection rule: lowest training loss across 3 seeds (a=seed0, b=seed1, c=seed2).

## Generation Parameters (Table 5.5)
- White-box surrogate: YOLOv8n
- Patch size: 80×80 pixels (~1.6% of 640×640 image area)
- Optimiser: Adam, lr=0.03
- Iterations: 250 per seed
- Seeds: {0, 1, 2}; best = lowest training loss
- EOT transforms applied at each step
